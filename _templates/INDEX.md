# _TEMPLATES — Maven's Template Library

> Pattern-match these for new entries. Consistency makes downstream analysis possible.

---

## Available Templates

- [[daily-note]] — Daily operational note (morning brief + end-of-day sync)
- [[session-log-entry]] — Append to [[SESSION_LOG]] at session end
- [[decision-entry]] — Load-bearing decisions → [[DECISIONS]]
- [[mistake-entry]] — Errors + root cause → [[MISTAKES]]
- [[pattern-entry]] — Validated approaches → [[PATTERNS]]
- [[skill-template]] — New skill creation
- [[campaign-brief]] — New campaign launch prep (canon-cited)
- [[content-registry-entry]] — New piece registered for attribution

---

## How To Use

Copy the template content, fill in placeholders (`{{date}}`, `{{title}}`, etc.), paste into the destination file.

Every template includes a `## Obsidian Links` section — KEEP it when pasting. That's how graph connections survive.
