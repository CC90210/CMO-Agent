## How to Read the Canon

### For a specific task (ad, funnel, content piece):
1. Which 2-3 frameworks apply most?
2. Read those files.
3. Cite them in the deliverable's "Canon applied" line (see [[WRITING]]).

### For self-improvement (general knowledge extension):
- Start with [[MARKETING_CANON]] (the 10 pillars summary).
- Then pick the deep-dive cluster relevant to current campaign work.
- Re-read one canon file weekly as drift-prevention.

### When stuck on a copy/strategy problem:
- [[ritson-diagnosis]] — diagnose the layer the problem lives at (research / strategy / tactics / execution).
- Then consult the canon file for that layer.

## Adding to the Canon

New canon files earn a spot when:
1. The framework has survived 5+ years of evidence
2. Cited by 2+ existing canonical voices
3. Resists fashion
4. Has produced repeatable results in Maven's campaigns

Propose additions via `proposals/` + [[PATTERNS]] entry. Promote after 3 successful applications.
